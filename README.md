# Sobre

<p>
Âmago é uma plataforma que tem como objetivo facilitar o acesso as ONGs, ampliando o canal de comunicação e expondo as
suas mercancias através de uma plataforma web, com a finalidade de promover um
melhor alcance de visualizações e recursos monetários.

 > Objetivos específicos:

a) Desenvolver um sistema web capaz de oferecer um método dinâmico
de compra e venda de produtos, oferecidos por ONGs;

b) Divulgar os projetos propostos pelas ONGs;

c) Expor a localização de ONGs, facilitando uma maior participação da
sociedade para com elas;

d) Criar uma opção de arrecadação facilitada de doações, para as ONGs;

e) Desenvolver um sistema prático para incentivar o voluntariado.
</p> 

# Ferramentas Utilizadas

<p>

> INTERFACE GRÁFICA

Desenvolvimento do design feito através das tencologias: Illustrator,
Photoshop.

> ESTRUTURAÇÃO

Desenvolvimento do front-end feito através das tencologias: WordPress, Bootstrap, HTML e CSS.

Desenvolvimento da estrutura feito através das tencologias:  Node JS, JavaScript e MySQL.
Fora utilizado também as bibliotecas Express, Jquery, Multer e Md5 para
aprimoração da plataforma.

</p>
